Разрабатывай Telegram Mini App для коллекционирования футбольных карточек: игровая валюта, паки с разными шансами редкости и поэтапной анимацией открытия, коллекция, мини-игры, режим с использованием карточек, обмены между игроками и удобная админ-панель.
Используй расширяемую архитектуру, серверную игровую логику и безопасные транзакции; стек: React + TypeScript для frontend, FastAPI + PostgreSQL для backend, aiogram 3 для Telegram-бота, Docker для запуска проекта.
